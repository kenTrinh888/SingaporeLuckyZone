Change Marker

1.     Choose Layer from drop down menu

2.     Choose Color of icond holder in dropdown menu:

3.     Choose icon display on the holder. E.g: car, coffee, etc

The list of icon can be found at: https://fortawesome.github.io/Font-Awesome/icons/

 Display Layer

 

User can toggle different players to show specific layer. Sometimes layers need to be display in order. For example, to display the Propotional Wins above zone layer, we need to uncheck Propotional Wins Layer and Zone layer. Then check Zone Layer first, then propotional Wins Layer
